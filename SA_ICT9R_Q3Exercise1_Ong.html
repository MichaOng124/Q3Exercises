My First Bootstrap Page

HTML

Hyper Text Markup Language

Hyper Text Markup Language or HTML in short is used for creating and organizing web pages. It uses tags and elements to add headings, paragraphs, and tables.

CSS

Cascading Style Sheets

Cascading Style Sheets- CSS- is used to add more details to a web page, and make the site look more interesting. Using CSS you can alter fonts and its sizes, typefaces, etc. You can also add colors and resize images. The three kinds of CSS are inline, internal, and external.

Bootstrap

Bootstrap Version 4

Bootstrap uses responsive web design to rezise the contents of a webpage to fit all screen sizes. This way, you do not have to use scroll bars to view the length and width of the page.
